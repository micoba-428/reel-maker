# 🎬 Reel Maker

写真アプリや端末内の写真をPoolへ追加して、Instagramリール（縦動画）を自動生成するアプリ。

## 機能
- 📥 写真アプリ/端末内写真からPoolへ直接追加（pool_1〜pool_5）
- 🎬 パターン選択でリール自動生成（1-2-3-4-5 / 1-2-3-1）
- ☁️ Google Drive連携（写真・出力先）
- 📱 モバイル対応UI

## 使い方
1. アプリで追加先のpool 1〜5を選択
2. 写真アプリ/端末内の写真を選んでPoolへ追加
3. パターン・長さ・タイトルを選んでリール生成
4. 出力されたMP4をInstagramに投稿
